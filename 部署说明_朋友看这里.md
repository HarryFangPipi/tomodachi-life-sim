# 友达世界 — 部署说明（给朋友）

一个开罗风格的像素小镇，居民会自己走动、聊天、积累记忆和关系。
对话由**本机的 Ollama 大模型**驱动。

---

## 一、需要先装的东西

### 1. Python（3.10 或更高）
- 官网下载：https://www.python.org/downloads/
- 安装时**勾选 "Add Python to PATH"**。

### 2. Ollama（AI 大脑，必须）
- 官网下载安装：https://ollama.com/download
- 装好后打开终端，拉一个模型（二选一）：
  ```bash
  ollama pull qwen2.5vl:7b      # 效果好，约 5.6GB，建议有点显存/内存的机器
  ollama pull llama3.2:3b       # 更小更快，约 2GB，配置一般也能跑
  ```
- > 不装 Ollama 游戏也能开，但角色只会说固定的套话，不会真正"思考"。

---

## 二、运行

### Windows
双击 **`start_server.bat`**（会自动装依赖并启动），然后浏览器打开：
```
http://localhost:8000
```
> 玩的时候**不要关掉那个黑色命令行窗口**，关了服务器就停了。

### Mac / Linux
```bash
pip install -r requirements.txt
python server.py
```
然后浏览器打开 `http://localhost:8000`（具体端口看 `config.json` 里的 `port`）。

---

## 三、配置（`config.json`）

打开 `config.json`，把 `preferred_models` 列表的**第一项改成你 `ollama pull` 下载的模型名**，例如：
```json
"preferred_models": ["llama3.2:3b", "qwen2.5vl", "qwen3"]
```
其它常用项：
- `port` —— 网页端口（默认 8000，被占用就改成 8001 等）
- `ollama_url` —— 默认 `http://localhost:11434`，本机跑 Ollama 不用改
- `tick_seconds` —— 游戏节奏，越小越快

---

## 四、常见问题

- **角色只说套话 / 不对话** → Ollama 没开，或 `config.json` 里的模型名和你下载的对不上。
- **端口被占用** → 改 `config.json` 的 `port`。
- **想从头开始** → 删掉 `data/savegame.json`（没有就是全新小镇）。
- **加新居民** → 网页右上角"＋ 新居民"，会自动生成带描边的像素小人，数据自动保存。
